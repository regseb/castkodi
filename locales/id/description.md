Cast&nbsp;Kodi juga menganalisis halaman untuk video, musik atau integrasi platform eksternal. Sebagai contoh, jika sebuah halaman menampilkan video YouTube, video tersebut akan dikirim ke Kodi.

Untuk meng-cast video / musik, tiga item dapat ditambahkan pada menu konteks dari tautan / halaman / pilihan video atau audio: _Putar sekarang ke Kodi_, _Putar berikutnya ke Kodi_ dan _Antrikan item ke Kodi_. Sebuah remote (dapat diakses dengan tombol toolbar) juga tersedia untuk meng-cast tab saat ini ke Kodi dan untuk: jeda, ke elemen berikutnya, mengatur volume, …

Untuk menghubungkan ekstensi ke Kodi, Anda harus _Mengizinkan kendali jarak jauh dari aplikasi di sistem lain_. Opsi ini ada di halaman _Pengaturan_ / _Layanan_ / _Kendali_ di Kodi. Anda juga harus mendapatkan _alamat IP_ Kodi. Informasi ini ditampilkan di halaman _Pengaturan_ / _Informasi sistem_ / _Ringkasan_. Lalu masukkan alamat IP ini di _Opsi_ Cast&nbsp;Kodi.
